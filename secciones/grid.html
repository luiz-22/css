Propiedades del padre

display

Esto define un contenedor de cuadrícula; en línea o en bloque según el valor dado. Establece un contexto de cuadricula para todos sus hijos directos. 

            
.grid-container {
  display: grid; /* or inline-grid */ 
}

grid-template-columns
grid-template-rows

.grid-container {
  display: grid;
  grid-template-columns: 50% 100px 1fr;
  grid-template-rows: 30% 100px 1fr;
}

Flujo de la grid

  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: 40px;

  grid-auto-rows: 80px;

justify-items
align-items

Alinea los elemento, respecto de su celda.
justify-items: alinea los elementos en el eje horizontal (X).
align-items: alinea los elementos en el eje vertical (Y). 

            
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 70px);
  grid-template-rows: repeat(3, 70px);
  justify-items: center; /* start | end | center | stretch */
  align-items: start; /* start | end | center | stretch; */
}

justify-content

justify-content: alinea los tracks de la grid en el eje horizontal (X),

  align-content: space-between; /* start | end | center | stretch | space-around | space-between | space-evenly */

justify-self

Alinea los elemento, respecto de su celda. Se puede aplicar a un elemento en particular.

  align-items: start; /* start | end | center | stretch; */
}

.grid-container .item:nth-child(4) {
  justify-self: start;
  align-self: end;

Grid dinámica

auto-fill: rellena la grid con tracks generados dinámicamente.
auto-fit: ajusta la grid con los tracks existentes.
En auto-fill, pueden quedar celdas vacias, en auto-fit, no. 

            
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fill, 100px);
  /* grid-template-columns: repeat(auto-fit, 100px); */ 
}

auto-fit: dice que se van a repetir tantas columnas como lo necesite
