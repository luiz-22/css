Propiedades del padre

display

Esto define un contenedor flexible; en línea o en bloque según el valor dado. Establece un contexto flexible para todos sus hijos directos. 

            
.flex-container {
  display: flex; /* or inline-flex */ 
}

flex-direction

Esto establece el eje principal, definiendo así la dirección en que se colocan los elementos en el contenedor. 

            
.flex-container {
  display: flex;
  flex-direction: row; /* row-reverse | column | column-reverse; */
}

flex-wrap

wrap: los elementos flexibles se envolverán en varias líneas, de arriba a abajo. 

            
.flex-container {
  display: flex;
  flex-wrap: wrap; /* nowrap | wrap-reverse */
}

justify-content

Define la alineación de los elementos hijos, respecto del eje principal. 

            
.flex-container {
  display: flex;
  flex-wrap: wrap;
  justify-content: space-between;
  /* flex-start | flex-end | center | space-between | space-around | 
  space-evenly | start | end | left | right ... + safe | unsafe */
}

2
3

align-items

Define la alineación de los hijos en el eje transversal (cross axis), dentro de cada línea (con wrap se forman varias líneas ) 

            
.flex-container {
  display: flex;
  flex-wrap: wrap;
  align-items: center;
  /* stretch | flex-start | flex-end | center | baseline | first baseline | 
  last baseline | start | end | self-start | self-end + ... safe | unsafe */
}

align-content

Define la alineación de los hijos en el eje transversal (cross axis), NO funciona cuando los hijos están en UNA sóla línea (es decir cuando flex-wrap tiene el valor de nowrap, NO FUNCIONA) 

            
.flex-container {
  display: flex;
  flex-wrap: wrap;
  align-content: space-around;
  /* flex-start | flex-end | center | space-between | space-around | space-evenly | 
  stretch | start | end | baseline | first baseline | last baseline + ... safe | unsafe */
}

Propiedades del hijo

order

Representa el orden que tendrán los elementos hijos en la caja flexbox. 

            
.flex-container {
  display: flex;
  flex-wrap: wrap;
}
                  
.item5 {
  order: -1;
}
                  
.item2 {
  order: 5;
}

flex-grow

Cuando la caja flexbox tenga espacio sobrante, es la habilidad o el factor de crecer de los hijos, valor por defecto es 0, NO se aceptan valores negativos. El espacio sobrante que se reparten los hijos, 1, 2, 3, ... son las proporciones que se reparten. Ej: Sobran 100px | item1 = 1 - item2 = 2 - item3 = 1 | item1 = 25px - item2 = 50px - item3 = 25px | px que se les agregan. 

            
.flex-container {
  display: flex;
}
                  
.item {
  flex-grow: 1;
}
1
2

flex-shrink

Cuando la caja flexbox NO tenga espacio sobrante, es la habilidad o el factor de encogerse, valor por defecto es 1, NO se aceptan valores negativos. El espacio que falta se les reduce a los hijos, 1, 2, 3, ... son las proporciones que se le quita. Ej: Faltan 100px | item1 = 1 - item2 = 2 - item3 = 1 | item1 = 25px - item2 = 50px - item3 = 25px | px que se les quitan. 

            
.flex-container {
  display: flex;
}
                  
.item:nth-child(2) {
  padding: 0;
  flex-shrink: 2;
}
  
.item:nth-child(4) {
  padding: 0;
  flex-shrink: 3;
}

flex-basis

Es el tamaño del elemento hijo dentro de la línea de la caja flexbox. Si la caja flexbox tiene dirección de fila, flex-basis representa el width. Si la caja flexbox tiene dirección de columna, flex-basis representa el height. Valor por defecto auto. Si el valor es 0, el espacio adicional alrededor del contenido no se tiene en cuenta. 

            
.flex-container {
  display: flex;
}
  
.item {
  flex-basis: 100px;
}

2

flex

Es un shorthand de las propiedades flex-grow, flex-shrink y flex-basis. 

            
.item {
  flex: none; /* flex-grow [flex-shrink] || [flex-basis] */
}

align-self

Sobreescribe el valor de la propiedad align-items sólo para el elemento hijo especificado. 

            
.flex-container {
  display: flex;
  align-items: center;
}
  
.item {
  align-self: flex-end; /* auto | flex-start | flex-end | center | baseline | stretch */
}
